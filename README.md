# 🐍 Python Open Source Software Project

## 📌 Project Overview

This project presents a detailed study of **Python as an Open Source Software (OSS)**. It includes both theoretical concepts and practical implementation using Linux through **Windows Subsystem for Linux (WSL)**.

The project explores Python’s origin, licensing, ecosystem, and practical usage in a Linux environment along with shell scripting.
